# Voice-bot
 🎙️ Simple Voice Bot

A simple AI Voice Bot project built using Python.  
This bot can take voice input from the user and respond with speech output.

## 🚀 Features
- Voice input using microphone
- Speech-to-text conversion
- Text-to-speech response
- Simple chatbot interaction
- Beginner-friendly project

## 🛠️ Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- PyAudio
